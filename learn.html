How to Keep Writing This Adventure

This adventure is a number of web-pages linked together.

To finish it, you'll need to change the existing web-pages and add some new ones.

What a Web-page is

A web-page is a special type of document. It's written in HTML: HyperText Markup Language.

A web-page can contain different 'elements': like headings, paragraphs, images and links to other web-pages.

You define what elements should appear in the web-page using HTML.

When you open the web-page in your browser, the browser reads the page's HTML, and shows the defined elements.

Challenge 1: update a web-page to add your name

It's time to change a web-page in the adventure.

Let's update the adventure 'start' web-page.

It's your adventure, so we'll change the page to say 'by Your Name', rather than 'by Greg'!

First of all, you need to open the start web-page in an HTML editor program (ask a mentor if you need help):

  1. Sublime Text is a good editor. You can install it from http://www.sublimetext.com/3.
    • If you don't have permission to install new software, try Notepad on Windows.
  2. Now find the start.html file on your computer, and open it with the HTML editor.
    • The start.html file should be in the 'coderdojo-adventure-master' folder you downloaded and opened.

Web-page structure

In your HTML editor, The 'start' web-page HTML looks like this:

<html> <head> <title>CoderDojo - Choose Your Own Adventure</title> </head> <body> <h1>Choose Your Own Adventure</h1> <h2>By Greg</h2> <p>You are in a room.</p> <p>In front of you are some doors.</p> <p>You decide to: <ul> <li><a href="no-chickens.html">Go through</a> the door marked 'No Chickens Allowed'</li> <li><a href="secret-tunnel.html">Go through</a> the door marked 'Secret Tunnel'</li> </ul> </p> </body> </html>

The HTML defines the different 'elements' that make up the page: like headers, paragraphs and links.

Each 'element' is made up of a start 'tag' and an end 'tag', and some text in between.

See this line?

<h2>By Greg</h2>

This is HTML for 'header' ('h2' is for header of size '2'). This tells the web-browser to display the text 'By Greg' as a heading, in big, bold text. Like this:

By Greg

Now change this 'h2' to have your name, rather than 'Greg'. So if you were called Tim, it would be:

<h2>By Tim</h2>

Now save the start.html in your HTML editor, and reload the same file in your web browser.

If you were called Tim, the start page should now say:

by Tim

You just changed your first HTML webpage. Nice!

Challenge 2: create a new web-page

Let's add another door to the 'start' web-page of your adventure. It will point to a new web-page you can edit:

  1. Make sure the 'start.html' file is open in your HTML editor again.
  2. Now, find this line:<li><a href="secret-tunnel.html">Go through</a> the door marked 'Secret Tunnel'</li>
  3. Add this line right after: <li><a href="haunted-house.html">Go through</a> the door marked 'Haunted House'</li>
  4. Save the start.html file in your HTML editor, and reload the same file in your web browser.
  5. There should be a new line at the bottom of the web-page, about a Haunted House.
  6. Click on the 'Haunted House' link on that line.

Oh no! Things have gone wrong in your web browser.

Either the browser is showing an error message, or everything is blank.

What's the problem? Well, you just added some HTML that creates a link to a new web-page called 'haunted-house.html' (that's the 'a href=haunted-house.html' bit)

Your browser is complaining, as the haunted-house.html web-page doesn't exist yet.

Now it's time for you to create the 'Haunted House' web-page:

  1. Look in the 'coderdojo-adventure-master' folder where all your adventure files are.
  2. Find a file called new-webpage.html
  3. Copy that file, and paste it into the same folder, making sure the new file is called haunted-house.html
  4. Now reload your web browser (which should still be trying to open haunted-house.html)
  5. You should now be seeing a new web-page. If not, ask a mentor for help.

You have just created your first web-page. Well done!

Below is a list of HTML elements you can use.

Different HTML elements

- Paragraph -

A paragraph is some text with space before and after.

Here's an example paragraph in HTML:

<p>'You are in a room.'</p>

And here's what it looks like when loaded in the web-browser:

'You are in a room.'

- Header -

A 'header' element is a heading: it shows text in big and bold.

Here's an example header in HTML:

<h1>I am a header</h1>

And here's what it looks like when loaded in the web-browser:

I am a header

There are different sizes of header: try h1, h2, h3, ...

- Lists -

The list element is useful for writing a list of choices in your adventure.

Use the 'ul' element for that. It's a bit different, because each item in the list is also an element: called 'li' (for 'list item')

Here's an example list in HTML:

<ul> <li>Eat the rat</li> <li>Throw the rat</li> </ul>

And here's what it looks like in the web-browser:

  • Eat the rat
  • Throw the rat

- Links -

So you want to link to another web-page in the adventure?

You'll need the 'a' element for that (no idea why it's called 'a')

Here's an example link in HTML:

<a href="secret-tunnel.html">Enter the secret tunnel</a>

And here's what it looks like when loaded in the web-browser:

Enter the secret tunnel

Challenge 4: Adding pictures

Your adventure would probably look cooler if it had pictures.

Go and add some.

Use the 'img' element for that. It's a lot like the 'a' link element: but you're linking to a picture rather than another web-page.

Here's an example image in HTML:

<img src="robot.png"></img>
