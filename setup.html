CoderDojo: Write Your Own Adventure

  1. You will be editing HTML files today, so start by downloading a good HTML editor program: http://www.sublimetext.com/3
  2. Download a zip file from: http://tiny.cc/coderdojo-adventure
  3. Unzip the zip file, which creates a folder called: coderdojo-adventure-master (ask a mentor if you need help)
  4. In the coderdojo-adventure-master folder, open start.html in a web-browser
  5. Do what it says :-)
