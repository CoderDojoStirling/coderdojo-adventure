Improving Your Adventure

This session, you're going to improve the adventure you created last time.

First, you need to learn more about HTML and CSS.

You used HTML to create web-pages in your adventure.

You will use CSS to make the pages look nice.

Khan Academy Course

Go do the Khan Academy course called 'Hour of Webpages':

https://www.khanacademy.org/computing/hour-of-code/hour-of-html

The course takes an hour or less to do.

If you get stuck, then ask a mentor.

HTML Challenges

So you finished the course on HTML?

Let's see how good you are.

Make sure you have your adventure on your computer.

Don't have it? Download a new copy from http://tiny.cc/coderdojo-adventure

Make sure your adventure web-pages contain lots of HTML:

  1. 2 or more headers, using:

    <h1> - <h6>
  2. 2 pieces of important text, using: <strong>
  3. 2 pieces emphasised text, using: <em>
  4. 2 or more line breaks, using: <br>
  5. 2 or more paragraphs
  6. 2 or more hyperlinks to other pages
  7. 2 or more images with alt, width and height attributes set
  8. An ordered list
  9. An unordered list

And make sure you've got some CSS:

  1. 2 or more webpages have background colors, using the background-color style
  2. Some headers and some paragraph have coloured text, using the color-style
Stuck? - ask a mentor for help.

CSS Challenges

Make sure your adventure contains some advanced css:

  1. One paragraph has id 'like-rain', and background color blue
  2. Two headers have class 'like-fire' and color red
